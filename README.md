## Instagram Clone with Laravel 8 / Inertia JS / Vue JS / Tailwindcss

## Installation

- [composer install]
- [npm install]
- [npm run development]

## After installation

- [copy .env-example  to .env and set your mysql host, user, pw and db]
- [php artisan migrate]
- [php artisan serve]
- [npm run watch]
